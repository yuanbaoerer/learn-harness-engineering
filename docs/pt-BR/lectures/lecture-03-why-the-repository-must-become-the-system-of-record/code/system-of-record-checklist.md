# Checklist de System of Record

Uma nova sessão do agente consegue descobrir o seguinte apenas pelo repositório?

- Qual produto está sendo desenvolvido?
- O que o aplicativo deve fazer para os usuários?
- Como a base de código está organizada?
- Como iniciar a aplicação?
- Como verificar a integridade/saúde da aplicação?
- Qual trabalho está atualmente em andamento?
- Quais padrões de qualidade são importantes?
